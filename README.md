Aladin's Calculator
Introduction
This project is a straightforward web-based calculator created by Dominic Burfeind. It allows users to perform basic arithmetic operations and was designed with a clean and responsive interface.

Files
The project consists of three main files:

index.html: The main HTML file that defines the structure of the calculator interface.
style.css: The CSS file that styles the HTML elements, providing a visually appealing layout.
script.js: The JavaScript file that contains the logic for the calculator, including functions for user interactions and calculations.
Usage
To use the calculator, open the index.html file in a web browser. The calculator interface provides buttons for numbers, operators, and special functions. The display area shows user input and results.
